# planning-app

This project is an attempt to make a Trello-like task management app.

Using Vue, Vuex, MongoDB (probably) and Express for backend.

To-do

1. Add user OAuth

API Definition
/:project
GET boards for a given project.
POST add new project.
DELETE remove project
